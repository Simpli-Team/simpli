# Installation

## System Requirements

- Node.js 18.0 or later
- npm, yarn, or pnpm

## Create a New Site

The easiest way to get started is with the create-simpli command:

:::tip[Quick Start]
Use your preferred package manager to create a new project.
:::

```bash
# npm
npm create simpli@latest my-docs

# yarn
yarn create simpli my-docs

# pnpm
pnpm create simpli my-docs
```

## Start Development Server

```bash
cd my-docs
npm run dev
```

Your site will be available at `http://localhost:5173`.
